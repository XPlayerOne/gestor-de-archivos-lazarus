📂 Gestor de Archivos Lazarus (Guía de Usuario)
¡Bienvenido a tu nuevo Gestor de Archivos! Esta aplicación ha sido diseñada para ser rápida, sencilla y funcionar de forma nativa tanto en Windows como en Linux.

🚀 ¿Qué puedes hacer con este programa?
1. Navegación Avanzada (Barra de Herramientas)
La barra superior ahora incluye un sistema de navegación completo:

◀ Atrás / Adelante ▶: Navega por tu historial de carpetas visitadas, igual que en un navegador web.
▲ Subir: Sube un nivel al directorio padre.
🏠 Inicio: Te lleva instantáneamente a tu carpeta personal de usuario.
🔄 Refrescar: Fuerza la actualización de la vista para mostrar cambios externos.
2. Exploración Inteligente
Panel Izquierdo (Árbol): Muestra la estructura jerárquica. Al cambiar de carpeta con el teclado o ratón, todo se sincroniza automáticamente.
Panel Derecho (Lista): Muestra archivos y carpetas con iconos nativos del sistema.
Barra de Ruta: Puedes escribir directamente una dirección y pulsar Enter para ir allí.
Barra de Estado: Te indica cuántos elementos hay en la carpeta actual y qué tienes seleccionado.
3. Funciones de Archivo (Clic Derecho)
Abrir: Ejecuta archivos o entra en carpetas (también con Doble Clic o Enter).
Copiar/Pegar: Sistema multi-archivo que permite mover datos entre carpetas.
Eliminar: Borra elementos con confirmación de seguridad y reporte de errores detallado.
Renombrar / Crear Carpeta: Gestión básica de archivos integrada.
4. Arrastrar y Soltar (Drag & Drop)
¡Funcionalidad completa! Selecciona archivos en la lista de la derecha y arrástralos sobre cualquier carpeta del árbol de la izquierda para copiarlos directamente.
